About

Candy City

This project was made for a school assignment. We were asked to develop a Facebook advertisement for a game. It could be any game, but it needed to be directed to children. Thinking on that, I decided to create a memory game called Candy City. Originally, the game was to be something similar to SimCity, but the time to develop it was limited. That is why the game came in a simple form such as a memory game. By the way, all images were created by me.
